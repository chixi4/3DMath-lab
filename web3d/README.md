# Steinmetz Web3D

Three.js implementation for the Steinmetz two-cylinder intersection animation.

## Run Locally

```bash
npm install
npm run dev
```

## Capture

Render keyframes:

```bash
npm run capture:keyframes
```

Render keyframes and the final `1080p60` video:

```bash
npm run capture:video
```

Generated files are written to `../output/steinmetz-web3d/` and are not tracked by Git.
