# Research Notes

## Differential Insert

### 叙事目标

在首个 hero disk 完成之后，不立刻让镜头移动或继续批量生成，而是插入一个简短的解释段：

1. 首个圆盘先完整成立。
2. 在二维母体上抽出一个更薄的 strip。
3. 按 `dx -> y -> dA` 的顺序解释这个微分面积元。
4. 解释结束后，其余切片再开始旋转成盘，镜头同时启动。

这条动线把数学逻辑和视觉逻辑对齐成：

`先看到一个盘` -> `理解它来自哪个面积元` -> `再看整个系统展开`

### 视觉处理

- 背景从纯白提到带一点紫意的编辑白。
- differential 段保持相机静止。
- hero disk 稍微退到二级，但不消失。
- strip 放在平面上、紧挨 hero disk 左侧的位置。
- 标注分三拍出现，避免一口气贴满信息。

### 节奏结果

- differential beat 独立成一段完整的说明镜头。
- 后续切片不再只是淡入，而是继续沿用“旋转成盘”的动作语法。
- 镜头运动延后到系统重新展开时再开始。

## Linked Rectangles

### 改动目标

后续版本把微分解释进一步改成“矩形链”系统，而不是单独插入一个说明段。
核心想法是让平面矩形和体积切片在同一条时间线上建立对应关系。

### 主要变化

- 在首个圆盘出现前，先在 `xy` 平面建立第一块矩形。
- 首个圆盘完成后，剩余矩形依次长出来。
- 所有矩形在后续体积生成中保留在画面里。
- 合成器负责矩形显现、遮挡关系和终态增强。

### 后续修正方向

这个矩形链版本在落地时重点修过几类问题：

- 平面 strip 顶边要贴着曲线，而不是平顶矩形。
- reveal 阶段不能发生位置漂移，矩形必须始终在最终位置出现。
- 终态帧里只保留矩形时，填充和线条要足够清楚。
- 黑底版本下，矩形和圆盘边缘都要重新平衡显色强度。
