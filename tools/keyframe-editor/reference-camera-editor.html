怎么用
  • phi / theta / gamma / zoom 控相机角度和缩放。
  • center x/y/z 控镜头中心点，也就是画面真正对准哪里。
  • origin shift / world scale 是这条场景自己的坐标系平移和缩放，之前它们是隐藏的。
  • theta offset / focal distance 会影响“正视图到底是不是你直觉里的正视图”以及透视强弱。
  • 参考帧 t 默认放在中段，你也可以自己挑更顺手的一帧来调。
  • 改完点“保存并刷新参考帧”，右边就会立刻更新成新视角。
